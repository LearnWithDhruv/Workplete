# Usage Guide

## Running the AI Employee

1. **Load Data:**
   - Command: `load data`
   - Description: Loads the data file (CSV, JSON, Excel) into the system.

2. **Clean Data:**
   - Command: `clean data`
   - Description: Cleans and preprocesses the loaded data.

3. **Run Analysis:**
   - Command: `run analysis`
   - Description: Runs various analysis algorithms on the cleaned data.

4. **Generate Report:**
   - Command: `generate report`
   - Description: Generates a report based on the analysis, including visualizations.
